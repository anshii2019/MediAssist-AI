# MediAssist AI 🏥🤖

MediAssist AI is an intelligent telemedicine virtual assistant designed to help users analyze symptoms and receive preliminary healthcare guidance.

## 🚀 Features
- Symptom-based analysis using AI
- Conversational chatbot (LLM-powered)
- Basic medical guidance suggestions
- Easy-to-use interface

## 🛠️ Tech Stack
- Python
- LangChain
- OpenAI (ChatOpenAI)
- Flask / FastAPI (if used)

## 🎯 Goal
To provide quick and accessible healthcare assistance using AI before consulting a doctor.
